## Hippotizer REST API module

### Configuration

This module supports Hippotizer version 4.7.1 or later. To enable control of the Hippotizer Engine, add the REST API component to the Hippotizer active show.

### Actions

#### LAYER

- Load Media
- Fade Level
- Load Preset
- Play Modes

#### MIX

- Fade Level
- Load Preset

#### PIN

- Set Value
- Fade To Value
- Reset Value

#### TIMELINE

- Trigger Action
  Global Timecode
