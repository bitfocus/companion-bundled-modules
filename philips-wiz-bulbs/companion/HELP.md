## Philips Wiz Lightbulbs

This module gives you control of the Philips Wiz Lightbulbs.

## Supported Actions and Feedbacks

| Category | Name | Description |
|----------|------|-------------|
| Actions | Set Brightness (Will NOT Turn On Bulb) | Sets the bulb brightness |
| Actions | Set Color (Will Turn On Bulb) | Sets the bulb to a specific RGB color |
| Actions | Set Color Temperature (Will Turn On Bulb) | Sets the bulb to a specific color temperature |
| Actions | Set Scene (Will Turn On Bulb) | Sets the bulb to a predefined scene |
| Actions | Turn Bulb Off | Turns the bulb off |
| Actions | Turn Bulb On | Turns the bulb on |
| Feedback | Bulb On/Off State | Indicates whether the bulb is currently on or off |
| Feedback | Current Brightness | Indicates whether the bulb is currently at a specific brightness |
| Feedback | Current Color | Indicates whether the bulb is currently set to a specific RGB color |
| Feedback | Current Color Temperature | Indicates whether the bulb is currently at a specific color temperature |
| Feedback | Current Scene | Indicates whether the bulb is currently set to a specific scene |
