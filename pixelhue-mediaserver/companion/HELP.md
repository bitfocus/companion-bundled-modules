## Pixelhue mediaserver

This module will allow you to control the following Pixelhue media server products: MS3. MS3 Pro

### Configuration

- Enter the IP address of the product in the configuration settings.

**Available actions:**

- Program
- Pause
- Play
- Stop
- FTB
- Volume
- Brightness
- PPT page up/down
- OFF Test PIC
- On Test PIC
- Prev Program
- Next Program
