## TinkerList Cuez Automator

This module will allow you to control TinkerList's Cuez Automator.

### Configuration

- The module makes HTTP requests over port 7070 and uses Websockets for updates.

### To use the module

Use a preset or add an action to a button and choose the action you wish to use.

**Available Actions:**

- Next
- Previous
- Next Trigger
- Previous Trigger
- Trigger Deck Button
- Trigger Deck Switch On
- Trigger Deck Switch Off
- Trigger Deck Switch Click
- Trigger Shortcut
- Start Timer
- Stop Timer
- Stop All Timers
- Select Project
- Unload Episode
- Select Episode
- Trigger Item
- Trigger First Item

**Available Variables:**

- Automator ID
- Automator Name
- Automator Version
- Automator State
- Project ID
- Project Title
- Project Pairing Expired
- Episode ID
- Episode Title
