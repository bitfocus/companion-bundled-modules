# Wyrestorm Matrix

This module will control WyreStorm MX-H2A, MXV and MX-KIT Series Matrix Switchers.

## Configuration

- IP Address
- Enable Verbose Logging

## Actions

- Select Input
- Select Output
- Route Input to Output

## Feedbacks

- Selected Input is Routed to Selected Output

## Variables

- Output [number] Current Input
