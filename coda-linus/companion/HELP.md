# CODA LINUS6.4
This module will control the Coda Audio LINUS6.4 amplifier.

[https://codaaudio.com/electronics/linus6-4/](https://codaaudio.com/electronics/linus6-4/)

## Actions
* Standby on/off
* Mute all on/off
* Mute channels 1-4 on/off

## Feedback
* Standby state
* Channel mute
* Mute all

## Presets
* Standby
* Mute all
