## Kommander Media Server
This module connects to Kommander Media Server

### Configuration

- Type in the IP address of the device.
- Type in the Port of the device.

### Available Actions

- CallPlan
- ChangePlanGroup
- ChangePlayStatus
- SoundCtrol
- PageTurn
- Screen On/Off
- ChangeBright&Contrast
- Output On/Off
- Master Switch
- Lock
