# National Weather Service API

This module will show alerts from the national weather service in the US using the weather.gov api.

## Configuration

- Latitude/Longitude or Zone ID
- Trigger on Event Types
- Enable Verbose Logging

## Actions

This module has no actions.

## Feedbacks

- Event Type exists in Zone

## Variables

- Latitude/Longitude (if entered)
- Zone ID

Per Event Type:

- ID
- Status
- Category
- Response
- Severity
- Certainty
- Urgency
- Area Description
- Sent Time
- Effeective Time
- Onset Time
- Expiry Time
- End Time
- Headline
- Description
- Instruction
- Web Link
- Sender Name

## Presets

No presets.
