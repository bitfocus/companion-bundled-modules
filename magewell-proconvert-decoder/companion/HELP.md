# Magewell Pro Convert Decoder

This module will allow you to control functions of a Magewell Pro Convert Decoder

## Configuration

- The IP Address of the device.
- The Username to login
- The Password to login

## Actions

### Set Video Config

#### OSD

- Show Title
- Show Tally
- Show VU Meter
- VU Meter Mode
- Show Center Cross
- Safe Area Mode
- Ident Mode
- Ident Text

#### Video Process

- Horizontal Flip
- Vertical Flip
- Deinterlace Mode
- Aspect Ratio Convert Mode
- Alpha Channel Display Mode

#### Video Source

- Auto Color Format
- Set Color Format
- Switch Mode

### Set Video Mode

- Width
- Height
- Interlaced
- Field Rate
- Aspect Ratio

### Set Audio Config

- Gain
- Sample Rate
- Channels

### Decoding

- Select Preset Channel to Decode
- Select NDI Source to Decode
- Set NDI Config
- Set Playback Config

## Feedbacks

- Show Channel is Selected
- Show NDI Source is Selected
- NDI is Connected

## Variables

- Summary Info (Device Name, Model, etc.)
- Video Config Info (OSD, Process, Source)
- Video Mode Info
- Audio Config Info
- Channel and Source Info
- Network Info
