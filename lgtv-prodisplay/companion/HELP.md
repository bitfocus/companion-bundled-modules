# LGTV Pro Display

This module controls LG's Pro Displays.

## Configuration

- IP/Port of Display (default of TCP port 9761)
- Set ID (0 - 99)
- Polling Rate
- Enable Verbose Logging

## Actions

- Power On/Off
- Set Input Source
- Screen Mute On/Off
- Volume Mute On/Off
- Set Volume Level
- Increase/Decrease Volume Level

## Feedbacks

- Power State
- Input Source is [x]
- Screen is Muted
- Volume is Muted

## Variables

- Power State
- Input Source
- Screen Mute State
- Volume Mute State
- Volume Level
