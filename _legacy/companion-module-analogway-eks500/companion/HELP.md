## Analog Way EKS500
This module connects to an Analog Way EKS500

### Configuration
* Type in the IP address of the device.
* Choose whether to communicate over TCP or UDP.
* The device will use port 10500 to communicate.

### Available actions
* Take
* Background LIVE
* PIP 2
* PIP 3
* Background Frame
* LOGO 1
* LOGO 2
* User Preset
