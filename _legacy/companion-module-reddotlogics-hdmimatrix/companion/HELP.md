## Red Dot Logics Matrix

This module will connect to Red Dot Logics 4k Matrix
supported Models: 
* RDL-MX4k88SNS
* RDL-MX4k1616NS

### Configuration
* Type in the device IP address and port (default of 23).

### Available actions
* change Inputs to Outputs
* Recall Preset
* Save Re Calls
* Change EDID on any Inputs 
* Power on and off remotely
