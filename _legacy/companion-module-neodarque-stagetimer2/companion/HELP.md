## StageTimer2 module

This module is for the StageTimer2 software. Most of the commands
you need to select the timer X first.


**Available commands for StageTime2**

* start
* stop
* reset
* enable
* disable
* Next entry
* Previous entry
* Set timer from index
* Set time on time
* Enter fullscreen
* Exit fullscreen
