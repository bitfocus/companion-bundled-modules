## Sonoran Coyote

This module will connect to any Sonoran Coyote.

**Available commands for Sonoran Coyote**

* End
* Pause
* Take
* Take Next
* Take Prev
* Select Preset
* Select Next Preset
* Select Prev Preset
* Reboot Coyote
* Shutdown Coyote
* Soft Reboot Coyote
* SeekTo x (in milliseconds)

Not implemented yet:

* Get Presets
* Get Timecode
* Set Hardware Mode
