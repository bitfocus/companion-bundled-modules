## d3/disguise (OSC)
This module provides default OSC controls to the d3/disguise server. Setup OSC transport within d3/disguise.

**Available commands for d3/disguise**

**Input Messages**
* Play
* Play to end of section
* Loop section
* Stop
* Previous Section
* Next Section
* Return to start
* Previous track
* Next track
* Track name
* Track ID
* Cue
* Fade up
* Fade down
* Hold
* Volume
* Brightness

**Output Messages**
* Heartbeat
* Track position
* Track name
* Track ID
* Current section name
* Next section name
* Section hint
* Volume
* Brightness
* BPM
* Play mode
