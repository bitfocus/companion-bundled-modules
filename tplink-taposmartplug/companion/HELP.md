# TP-Link Tapo Smart Plug

This module controls TP-Link Tapo Smart Plugs and Bulbs.

## Actions:

* Power On/Off/Toggle
* Set Bulb Brightness
* Set Bulb Color

## Feedbacks:

* Power On/Off

## Variables:

* Device ID
* FW, HW Version
* Model
* Mac Address
* HW, FW, OEM ID
* On Time
* Overheated State
* Nickname
* Location
* Latitude/Longtitude
* SSID/Signal Level/RSSI
* Power State

## Presets

* Power On/Off
